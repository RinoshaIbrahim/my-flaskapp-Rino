# Flask Demo

This is a small Flask demo project created for learning and demonstration purposes.

Contents:
- `app.py` - main Flask application
- `templates/` - HTML templates

How to run locally

1. Create a virtual environment and install dependencies (example):

   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   pip install -r requirements.txt

2. Run the app:

   python app.py

Notes
- Update git user.name and user.email locally before committing if you want correct author info.
